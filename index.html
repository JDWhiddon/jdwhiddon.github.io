LegalEase

This is a comprehensive legal practice management tool that I developed using the .NET MAUI framework. The back-end is developed using C# and the front-end uses C# and XAML. The intuitive graphical user interface enables legal professionals to efficiently track clients and their respective projects. LegalEase uses my self-made REST API to access, create, and update data stored in my Microsoft SQL Server database.

Discord Bot

This is a discord bot that is used to track the performance of any PC player for Tom Clancy's Rainbow Six Siege. It is built solely with Python, and uses the Requests and BeautifulSoup libraries to gather the data of the requested player. It can be used by any Discord server that the bot is invited to, and is simple and intuitive. Along with gathering player data, this bot can also automatically send patch note updates for Rainbow Six Siege.

LinkedIn Job Scraper

This tool is built using JavaScript and the Axios and Cheerio libraries. It is designed to streamline the process of gathering job postings from LinkedIn. Axios fetches the web content and Cheerio parses the HTML structure, extracting relevant information from LinkedIn job postings. By compiling the desired job postings into a convenient .CSV file format, this provides an efficient solution for organizing and analyzing job opportunities.
