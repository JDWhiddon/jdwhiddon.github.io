Pomodoro Timer

This Pomodoro timer app is designed for Garmin watches and is developed using the Monkey C programming language. This unique productivity tool offers an effective solution for time management. The app guides users through 25-minute work intervals, each followed by a 5-minute break. Additionally, upon completing 4 cycles, users are rewarded with a longer, 20-minute break. With its intuitive interface and compatibility with Garmin devices, this app provides an excellent opportunity to enhance work habits and achieve higher levels of efficiency. You can find it on the Garmin ConnectIQ store here!

Discord Bot

This Discord bot is designed to monitor the performance of any PC player in Tom Clancy's Rainbow Six Siege. Built entirely with Python, it utilizes the Requests and BeautifulSoup libraries to retrieve the data of the specified player. The bot is compatible with any Discord server to which it is invited, offering a simple implementation. Beyond fetching player data, it also has the capability to automatically share patch note updates for Rainbow Six Siege.

LinkedIn Job Scraper

This tool is developed using JavaScript with the Axios and Cheerio libraries. Its purpose is to simplify the procedure of collecting job postings from LinkedIn. Axios is used to retrieve web content, while Cheerio parses the HTML structure to extract pertinent details from LinkedIn job listings. By compiling the selected job postings into a user-friendly .CSV file format, this tool offers an effective solution for arranging and analyzing various job opportunities.
